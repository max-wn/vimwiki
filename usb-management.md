## USB management

### Links

1. https://wiki.archlinux.org/title/Udev
2. https://archlinux.org/packages/core/x86_64/usbutils/

### Dependencies

1. Для работы с USB-устройствами под Linux можно использовать различные
   программы и утилиты. Одна из самых популярных - `udev`, которая отвечает за
   обнаружение и автоматическую настройку устройств. С помощью `udev` вы можете
   настроить разрешения на доступ к USB- устройствам, а также задать
   определенные действия, которые будут выполняться при подключении или
   отключении устройств.
2. Перед началом настройки необходимо убедиться, что у вас установлен пакет
   `usbutils`, который содержит утилиты для работы с USB- устройствами.

### Permissions. Разрешение доступа к USB-устройствам.

Для того чтобы пользователи имели доступ к USB-устройствам, необходимо выполнить
следующие действия:

1. Создать группу пользователей, которым будет разрешен доступ к USB-устройствам:

`sudo groupadd usbusers`

2. Добавить себя в созданную группу:

`sudo usermod -aG usbusers yourusername`

3. Создать файл правил `udev` для USB- устройств:

`sudo nano /etc/udev/rules.d/99- usb.rules`

4. Вставить следующий текст в файл:

```
ACTION=="add", SUBSYSTEM=="usb",
ATTRS{idVendor}=="XXXX",
ATTRS{idProduct}=="XXXX", MODE="0664",
GROUP="usbusers"
```

Где `XXXX` необходимо заменить на коды производителя (idVendor) и устройства
(idProduct) вашего USB-устройства. Вы можете узнать эти коды, выполнив команду
`lsusb`.

5. Сохранить изменения и закрыть файл. После выполнения этих шагов пользователи
   из группы `usbusers` будут иметь доступ к USB устройствам. При подключении
   нового устройства оно будет автоматически распознано и доступно для
   использования.

## Mount

Чтобы узнать идентификатор USB-устройства, вам нужно выполнить команду в
терминале. Откройте терминал и введите команду `lsusb` На экране появится список
всех подключенных USB-устройств с их идентификаторами. Запишите или запомните
идентификатор вашего USB-устройства.

Для монтирования USB-устройства воспользуйтесь следующей командой:

```sh
sudo mount /dev/usb_device_id /media/usb
```

Вместо `usb_device_id` укажите идентификатор вашего USB-устройства, который вы
получили ранее. Обратите внимание, что команду следует выполнять от
имени пользователя с правами суперпользователя (root).
